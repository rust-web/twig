Twig Extension API

Extensions can define new behavior during the compilation process, via token parser and node visitor. They can also define new node types in the abstract syntax tree, like test, unary operator, binary operator, function, filter, global

See twig::extension::core for example implementations. Note: The core extension is not yet fully implemented (see [CHANGELOG][changelog]).
